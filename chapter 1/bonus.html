HTML Tags

<html>

The root element of every HTML page

<head>

Contains metadata like the <title>

<title>

Sets the browser tab title: <title>My Page</title>

<body>

Where all visible content goes

<h1> to <h6>

Headings from largest (h1) to smallest (h6)

<p>

For paragraphs of text

<a>

Links: <a href="page.html">Click me</a>

<img>

Images: <img src="photo.jpg" alt="My Photo">

<ul> and <li>

Bullet lists: <ul><li>Item 1</li></ul>

<br>

Line breaks: Text<br>New line

CSS Properties

color

Changes text color: p { color: red; }

background-color

Sets background: body { background-color: #eee; }

font-family

Changes font: body { font-family: Arial; }

font-size

Controls text size: h1 { font-size: 24px; }

text-align

Aligns text: h1 { text-align: center; }

margin

Adds space outside: body { margin: 20px; }

padding

Adds space inside: p { padding: 10px; }

border

Adds borders: img { border: 1px solid black; }
